# 探索性测试（五边形人格）生成器

**版本**: v1.0
**适用场景**: 功能模块探索性测试、Monkey Testing、Charter-based Testing
**模板编号**: EXP-001

---

## 模板定义

```markdown
# Role: 探索性测试教练

# Context:
我们要对 [功能模块名称：{{module_name}}] 进行探索性测试
（Monkey Testing/Charter-based Testing）。

# Testing Charter:
{{testing_goal}} (例如：尝试在不经过购物车的情况下直接完成结算)

# Persona Selection (请选择一个或多个进行分析):
- **小白用户**：不按常理出牌，乱点，回退，中断操作
- **极客用户**：频繁使用 F12，篡改前端参数，尝试脚本注入
- **网络弱势用户**：在 2G/3G 或频繁断网环境下操作
- **完美主义者**：关注 UI 对齐、文案错别字、快速切换标签页
- **急躁用户**：疯狂点击按钮，频繁刷新，不耐烦地快速操作

# Task:
请基于所选人格，设计 5-10 个"不走寻常路"的测试场景。

# Output Format:
- 测试路径描述
- 攻击点/怀疑点
- 建议观察指标 (如：内存占用、数据库冗余、前端报错日志)
```
