                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        # 边界值分析（BVA）生成器

**版本**: v1.0
**适用场景**: UI 输入框、API 参数校验、表单字段测试
**模板编号**: BVA-001

---

## 模板定义

```markdown
# Role: 高级软件测试架构师 (QA Architect)

# Context:
当前正在进行 [项目名称：{{project_name}}] 的系统测试。
我们需要针对特定字段的输入限制，利用"边界值分析法"设计测试用例，以捕捉极端情况下的潜在崩溃或逻辑错误。

# Input Data:
- 目标字段：{{field_name}}
- 数据类型：{{data_type}} (如：int, float, string, date)
- 约束范围：{{constraints}} (如：1-100, 长度 5-20, yyyy-mm-dd)
- 业务逻辑备注：{{business_logic}}

# Task:
请根据上述输入，输出一套标准的边界值测试矩阵。

# Constraints:
1. 必须覆盖：最小值 (min)、最小值减 1(min-1)、最小值加 1(min+1)、最大值 (max)、最大值加 1(max+1)、最大值减 1(max-1)
2. 包含非法值检测：空值、特殊字符、超长字符串、错误数据类型
3. 输出格式必须为 Markdown 表格
4. 包含预期的错误提示语建议

# Output Format:
| 用例编号 | 测试场景 | 输入值 | 测试类型 | 预期结果 | 优先级 |
| :--- | :--- | :--- | :--- | :--- | :--- |
```

## 使用示例

### 示例 1：年龄字段边界值测试

**输入变量**：
- `project_name`: 退休返聘系统
- `field_name`: 用户年龄
- `data_type`: 整数
- `constraints`: 18-65
- `business_logic`: 退休返聘系统，非此区间无法注册

**预期输出**：

| 用例编号 | 测试场景 | 输入值 | 测试类型 | 预期结果 | 优先级 |
| :--- | :--- | :--- | :--- | :--- | :--- |
| TC-BVA-01 | 最小值边界 | 18 | 正向边界 | 提交成功 | P0 |
| TC-BVA-02 | 最小有效值 +1 | 19 | 正向边界 | 提交成功 | P1 |
| TC-BVA-03 | 最小无效值 (min-1) | 17 | 负向边界 | 报错：年龄需在 18-65 岁之间 | P0 |
| TC-BVA-04 | 最大值边界 | 65 | 正向边界 | 提交成功 | P0 |
| TC-BVA-05 | 最大值 -1 | 64 | 正向边界 | 提交成功 | P1 |
| TC-BVA-06 | 最大值 +1 | 66 | 负向边界 | 报错：年龄需在 18-65 岁之间 | P0 |
| TC-BVA-07 | 空值校验 | null | 异常校验 | 报错：年龄不能为空 | P1 |
| TC-BVA-08 | 数据类型错误 | "abc" | 异常校验 | 报错：请输入有效数字 | P1 |
| TC-BVA-09 | 负数边界 | -1 | 异常校验 | 报错：年龄需在 18-65 岁之间 | P2 |
| TC-BVA-10 | 超长值测试 | 999999 | 异常校验 | 报错：年龄需在 18-65 岁之间 | P2 |
