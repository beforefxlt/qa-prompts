## 可接受风险
- **残留规则**：虽然脚本有 `Ctrl+C` 清理逻辑，但若系统掉电或应用崩溃（SIGKILL），规则仍会残留。README 已提供手动清理指令作为补救措施，风险可控。
- **性能评估**：在大流量（>1Gbps）下的损耗尚未量化。鉴于主要用于功能稳定性验证，该风险在初期阶段可接受。
